# OpenCV Beginner Course 🚀

This repository contains my practice code and notes as I learn the fundamentals of computer vision using OpenCV and Python. The course covers everything from basic image manipulation to video processing.

## 🛠️ Prerequisites & Setup

Before running the scripts, make sure you have Python installed, then install the required dependencies:

```bash
# Install OpenCV and its contribution modules
pip install opencv-contrib-python

```

## 📂 Project Structure

* `Photos/` & `Videos/` – Media assets used for testing.
* `read.photo.py` / `read.video.py` – Basic scripts for loading and displaying media.
* `rescale.photo.py` / `rescale.video.py` – Functions to resize and rescale frames.
* `Changing Live Webcam Resolution.py` – Adjusting capture properties for live video.
* `draw.photo.py` – Drawing shapes (rectangles, circles, lines) and writing text on images.

## 📝 What I've Learned So Far

* How to read images and video streams (including live webcams).
* How to handle window resizing, rescaling, and resolution changes to optimize performance.
* How to use OpenCV's drawing functions to create shapes and text overlays on frames.

---

*Follow my journey as I build a foundation in Computer Vision!*
